# kami · Newsprint Cheatsheet

Primary target: Chinese `one-pager`, `long-doc`, `letter`, and `slides`.

This skill keeps the stable production pipeline, but the layout and content structure must be native Newsprint: newspaper sheet, masthead, column grid, lede, sidebar facts, black rules, and sparse red editorial marks.

Use `references/design.md` for the print-document spec. `prompts.md` remains the original website-oriented prompt source.
