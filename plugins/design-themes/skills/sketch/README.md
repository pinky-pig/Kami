# Sketch

Chinese-first Kami document theme for hand-drawn sketch output.

The production path is the stable Kami HTML -> PDF and Python -> PPTX flow. The visible design system is prompt-native Sketch: warm paper, handwritten typography, wobbly borders, hard offset shadows, thumbtacks, tape, sticky notes, and slightly tilted composition.

For `slides`, the production path is now dual-output: `slides_spec.py` feeds both the PPTX renderer (`slides.py`) and the Slidev online deck renderer (`assets/templates/slidev/render_from_spec.py`).
