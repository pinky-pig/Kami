# Sketch Design Reference

## Visual System

Sketch is a hand-drawn print system. It should look like paper notes, pin-board cards, and quick brainstorming artifacts rather than polished software chrome.

## Core Tokens

- Paper: `#fdfbf7`
- Ink: `#2d2d2d`
- Muted paper: `#e5e0d8`
- Red marker: `#ff4d4d`
- Blue pen: `#2d5da1`
- Sticky note: `#fff9c4`
- Card: `#ffffff`

## Composition

- Use dotted paper texture on the page.
- Use wobbly border-radius values on every important panel.
- Add slight rotation to avoid rigid alignment.
- Use solid offset shadows to suggest cut paper layers.
- Use tape strips, thumbtacks, dashed rules, and scribble lines as accents.

## Document Patterns

### One-Pager

Taped hero card, side notes, three metric stickies, one proof card, one action card.

### Long Doc

Pin-board cover, side meta note, pinned TOC, then lined article sheet plus sticky-note evidence.

### Letter

Header board, recipient note, dashed subject strip, notebook body, evidence notes.

### Slides

Note wall with rotated cards, offset shadows, tape, tacks, and marker accents.
