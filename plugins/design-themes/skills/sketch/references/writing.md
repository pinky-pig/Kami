# Writing Reference

Sketch output should feel human, collaborative, and work-in-progress.

## Shared Rules

- Keep wording direct and conversational.
- Favor concrete facts, next steps, and observations over polished slogans.
- Use short labels and note-like fragments when helpful.
- Let the page feel like a working board, but keep the content readable.

## One-Pager

Use one title card, one subtitle, three metrics, one evidence card, and one action card.

## Long Doc

Use cover board -> article sheet -> proof notes. Each section should open with the key idea in one short sentence.

## Letter

Relationship -> evidence -> fit -> clear recommendation. The side notes should contain only facts the reader can verify.
