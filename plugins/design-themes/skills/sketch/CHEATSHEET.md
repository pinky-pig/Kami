# Sketch Cheatsheet

## Must

- Use warm paper background `#fdfbf7`.
- Use handwritten stacks: `Marker Felt` / `Bradley Hand` / `Noteworthy` + `LXGW WenKai` / `TsangerJinKai02`.
- Use wobbly radius values, not standard rounded cards.
- Use hard offset shadows only: `4px 4px 0 0 #2d2d2d`.
- Add tape strips, tacks, dashed rules, small rotations, and sticky-note yellow.

## Never

- Do not use soft blur shadows.
- Do not use flat modern cards with perfect corners.
- Do not overfill the page with too many boxes.
- Do not use corporate sans stacks as the main typography.

## Content Units

- One-pager: taped title card, side note, 3 sticky metrics, proof card, action card.
- Long doc: cover board, pinned TOC note, lined article sheet, side proof notes.
- Letter: taped header, recipient note, dashed subject strip, notebook body, evidence notes.
- Slides: paper board, pinned cards, scribbled connectors, note-wall rhythm.
- Slides content source: keep `slides_spec.py` as the single source for both PPTX and Slidev.
