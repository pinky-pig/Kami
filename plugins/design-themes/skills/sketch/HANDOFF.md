# Sketch Handoff

This skill now uses the Kami production pipeline but its own hand-drawn visual language.

What matters:

- Keep the paper texture visible.
- Keep borders irregular and slightly messy.
- Use hard offset shadows, not blur.
- Use handwritten fonts with Chinese handwritten fallback.
- Keep layouts breathable; the page should feel like a creative board, not a dense report grid.
- Keep the Slidev online deck in the same note-wall language as the PPTX; it is a deliverable, not a fallback preview.
