# Content Strategy

怎么写，不是怎么排。民国报纸风格会放大内容的“公共性”和“事件感”，所以文字要更像**报道、特刊、号外、通信剪报**，而不是现代营销稿。

---

## 核心原则

### 1. 新闻事实优于形容词

- 不写："带来了显著增长"
- 要写："三个月内从 12 个客户扩展到 47 个客户"

每个关键句都要能回答：谁、何时、在哪里、做了什么、结果如何。

### 2. 标题像报纸标题，不像 PPT 标题

标题要短、硬、有信息量：

- `家书抵万金`
- `战火纷飞，笔墨传情`
- `三个月完成首批试点`
- `从零散线索到统一台账`

避免：

- `项目背景与价值分析`
- `关于某某能力建设的阶段性思考`
- `赋能业务增长新范式`

### 3. 导语先给结论

报纸风格的第一段是 lead，不是铺垫。用 2-3 句讲清核心：

1. 发生了什么
2. 为什么重要
3. 读者接下来应该看什么证据

### 4. 栏目化表达

把长内容拆成栏目：

- `要闻`
- `见字速归`
- `一线记录`
- `数字简报`
- `人物小传`
- `编辑按`
- `后续行动`

栏目名要短，不要像现代导航。

---

## 按文档类型

### One-Pager · 头版 / 号外

目的：30 秒内让读者抓到“今天这版报纸在讲什么”。

结构：

1. **报头**：4-8 字，像报刊名或特刊名
2. **日期线**：日期、地点、期号
3. **主标题**：一句核心判断
4. **导语**：2-3 句
5. **剪报框**：3-5 个，分别放数据、证据、时间线、引用、行动
6. **落款 / 红印**：一处即可

写作规则：

- 总字数控制在 450-700 字
- 小标题要像栏目，不像现代 section
- 数据必须占正文 25%+
- 不要使用“近年来/随着/在...背景下”开头

### Long Document · 多页特刊 / 深度报道

结构：

1. **封面头版**：报头 + 大标题 + 日期线 + 摘要
2. **版面索引**：像目录，但命名为“本期索引/版面索引”
3. **要闻摘要**：3-5 条 takeaway
4. **正文报道**：每章一版，章节名像栏目
5. **附录**：资料来源、方法、参考

写作规则：

- 每章首段是报道导语
- 大段内容拆成 2 栏或剪报框
- 图表前用一句“读图说明”
- 用事实链组织内容：事件 -> 证据 -> 判断 -> 下一步

### Letter · 家书特刊 / 通信剪报

结构：

1. **竖排题名**：如 `家书抵万金`
2. **反白短句**：一句情绪或主题
3. **通信信息**：寄件人、收件人、日期、地点
4. **正文**：3-5 段，正式但有温度
5. **证据框**：推荐信/推荐函放 2-3 条具体事实
6. **落款红印**

写作规则：

- 推荐信按“关系 -> 证据 -> 适配 -> 明确推荐”写
- 不堆人品形容词，写可核验事实
- 家书可以更克制有情绪，但不要煽情
- 正文不要全部竖排

### Slides · 编辑部汇报

结构：

1. 封面像报头
2. 每页一个标题判断
3. 数据页像“数字简报”
4. 案例页像“现场剪报”
5. 结尾页像“后续行动”

## 文字口径

### 可用语气

- `本期记录`
- `据项目台账`
- `截至 2026.04`
- `编辑按`
- `一线回声`
- `后续行动`

### 少用/禁用

- `赋能`
- `打造`
- `新范式`
- `闭环`
- `抓手`
- `底座`
- `生态`

除非用户原文必须保留，否则改成人话。

### 数字格式

| 要 | 不要 |
|---|---|
| 5,000+ | 5000+ |
| 90% | 90 % |
| 2026.04 | 2026年4月 |
| 3-5 项 | 三到五项 |

### 引号

中文优先 `「」`，栏目名不用引号。

## 交付前自查

- [ ] 标题是否像报纸标题，而不是 PPT 标题？
- [ ] 第一段是否直接给结论？
- [ ] 栏目名是否短、硬、有辨识度？
- [ ] 是否有事实、数字、日期、人物、地点？
- [ ] 是否删掉了现代商业黑话？
- [ ] 红印/报头是否服务内容，而不是纯装饰？
