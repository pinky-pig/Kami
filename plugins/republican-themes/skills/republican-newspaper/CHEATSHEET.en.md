# kami · Republican Newspaper Cheatsheet

Quick reference. Full spec lives in `references/design.en.md`.

## Scope

- Official: Chinese `one-pager`, `long-doc`, `letter`, `slides`
- Mood: Republican-era newspaper, special issue, extra edition, clipping layout
- Legacy: English templates, resume, portfolio

## Rules

1. Use cream newsprint `#E8DFC9` and black ink `#161411`; no archive-blue frame
2. Emphasis comes from reverse black labels, vertical mastheads, and thin rules
3. The only strong accent is seal red `#B7352A`
4. Use serif for mastheads, headlines, and vertical titles
5. Dense columns are allowed, but keep line-height at `1.35+`
6. Use solid borders only; no `rgba()`
7. Images should be grayscale, low-contrast, and evidence-like
8. Use mastheads, datelines, clipping frames, postal marks, and one red seal; avoid poster clutter

| Role | Hex | Use |

| Newsprint | `#E8DFC9` | Page base |

| Seal Red | `#B7352A` | Stamp / seal only |

## Type

| Role | Size | Weight | Line-height |
|---|---|---|---|
| Masthead | 36-54pt | 500 | 1.05 |
| Vertical Title | 26-40pt | 500 | 1.10 |
| Headline | 18-28pt | 500 | 1.12 |
| Body | 9.2-10.2pt | 400 | 1.42-1.50 |
| Dense Body | 8.4-9.2pt | 400 | 1.35-1.42 |
| Caption | 7.5-8.5pt | 400 | 1.35 |

## Component Choices

| Need | Use |
|---|---|
| Big title | Masthead or vertical title, black ink |
| Section label | Black reverse label |
| Evidence | Narrow sidebar / clipping frame |
| Photo | Grayscale image with thin border |
| Date | Small dateline, horizontal or vertical |
| Accent | One red seal |

Guiding sentence: **make it feel like a credible old newspaper special issue, not a retro poster.**
